Incluir CSS em HTML é como escolher como vestir seu site: você pode colocar a roupa diretamente na peça (inline), preparar um conjunto no cabeçalho (interno), ou usar um guarda-roupa externo (arquivo .css). A melhor prática é o método externo, pois mantém o código organizado e facilita manutenções. Exemplo Prático: No HTML abaixo, o CSS é vinculado via < link >, e todos os estilos estão no arquivo .css, sem alterações no HTML. As palavras em < span > já existentes ganham destaque apenas via CSS!

Três Formas de Usar CSS:

Inline: Estilo direto na tag (ex: <p style="color:red">).
Interno: Usando <style> dentro do HTML.
Externo: Linkando um arquivo .css (como este exemplo!).
