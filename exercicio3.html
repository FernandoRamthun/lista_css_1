Incluir CSS em HTML: Como Vestir Seu Site


Incluir CSS em seu site é como escolher a roupa que o seu site vai usar. Você pode adicionar o estilo de três formas principais: diretamente nas tags (inline), dentro do próprio código HTML (interno), ou ainda usar um arquivo separado, o CSS externo, que é o mais recomendado para manter seu site organizado e fácil de atualizar.

Três Formas de Usar CSS

Inline: O estilo é aplicado diretamente na tag, usando o atributo style. Por exemplo, <p style="color:red">.
Interno: O estilo é inserido dentro da tag <style> na seção <head> do documento HTML.
Externo: O CSS é colocado em um arquivo separado com extensão .css, e o arquivo é vinculado ao HTML com a tag <link>.

CSS Externo

O CSS externo é o método mais eficiente e escalável, ideal para sites grandes ou com várias páginas. Ele permite que você defina todos os estilos em um único arquivo e os reutilize em várias páginas HTML. A inclusão do CSS é feita com a tag <link> dentro do <head>.

CSS Interno

Quando você precisa de um estilo único para uma página específica, pode usar o CSS interno. Para isso, o código CSS é colocado dentro da tag <style> na seção <head>.

Figura 2 - Exemplo de CSS Interno

CSS Embutido

O CSS embutido é aplicado diretamente a um elemento específico, através do atributo style. Isso pode ser útil para estilizar um único elemento sem alterar o estilo geral da página.

Figura 3 - Exemplo de CSS Embutido

Ordem de Procedência

Quando você utiliza mais de um tipo de CSS em uma página, a ordem de inclusão é importante. O CSS embutido tem a maior prioridade, seguido pelo CSS interno e, por último, o CSS externo. Ou seja, se houver conflito, o último estilo aplicado prevalecerá.
