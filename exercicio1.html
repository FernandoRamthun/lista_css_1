O que é uma Regra CSS?

Uma regra CSS é um conjunto de instruções que informam ao navegador como os elementos HTML de uma página devem ser exibidos. A regra CSS altera características como cores, fontes, tamanhos e posicionamento de elementos na página.

Estrutura de uma Regra CSS

Uma regra CSS é composta por duas partes principais:

  • Seletor: Define o ou os elementos HTML aos quais a regra se aplica. Pode ser uma tag HTML, uma classe, um ID ou uma combinação deles.
  • Declaração: Define o que será estilizado. A declaração é composta por uma ou mais propriedades e valores.

O formato básico de uma regra CSS é:

        seletor {
            propriedade: valor;
        }

Exemplo Prático de Regra CSS

Aqui está um exemplo de uma regra CSS que altera a aparência de um título h1:

        h1 {
            color: #2c3e50;  /* Cor do texto */
            text-align: center; /* Alinha o texto ao centro */
            font-family: Arial, sans-serif; /* Define a fonte */
        }

Este código fará com que todos os títulos h1 da página tenham a cor azul escuro, sejam centralizados e usem a fonte Arial.

Exemplo de Funcionamento

Para testar a regra, você pode criar um arquivo CSS externo e aplicá-lo ao seu HTML. O exemplo abaixo mostra um código HTML com um arquivo CSS externo.

HTML (`index.html`)

        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Exemplo de Regras CSS</title>
            <link rel="stylesheet" href="style.css">
        </head>
        <body>
            <h1>Exemplo de Título com CSS</h1>
            <p>Este é um parágrafo com a formatação aplicada através de CSS externo.</p>
        </body>
        </html>

CSS (`style.css`)

        h1 {
            color: darkblue;
            text-align: center;
            font-family: 'Arial', sans-serif;
        }

        p {
            font-size: 16px;
            color: #333;
        }

Com esses arquivos, você verá que o título ficará em azul escuro e o parágrafo terá um tamanho de fonte de 16px e cor #333.

Importância da Cascata (Cascading)

A palavra "Cascading" no CSS refere-se à maneira como os estilos são aplicados e resolvem conflitos. Se várias regras forem aplicadas a um mesmo elemento, o navegador vai seguir a ordem de prioridade das regras, com base na especificidade e na ordem do código.

Outros Tipos de CSS

O CSS pode ser aplicado de diferentes maneiras:

  • Inline: A regra é aplicada diretamente em um elemento HTML usando o atributo style.
  • Interno: A regra é escrita dentro da tag <style> no cabeçalho do documento HTML.
  • Externo: A regra é colocada em um arquivo CSS separado e vinculada ao HTML usando a tag <link>.
