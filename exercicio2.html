Imagina que você está decorando uma parede com vários quadros. Cada quadro tem um estilo único: moldura diferente, cor específica e detalhes em destaque. Uma regra CSS funciona assim: você define "instruções" (como moldura, cor) para cada elemento da página, sem alterar a estrutura original (a parede). No exemplo abaixo, cada parágrafo é um "quadro" com fonte e cor únicas, e palavras marcadas com < span > ganham efeitos especiais apenas via CSS (sem mudar o HTML!).
