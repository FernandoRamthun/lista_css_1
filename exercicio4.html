Entendendo a Especificidade no CSS

O CSS segue um sistema de hierarquia chamado "especificidade" para decidir quais regras têm prioridade sobre outras. Abaixo, exploramos 8 regras diferentes e como elas afetam a estilização de uma lista.

Seja o trecho de código HTML e CSS da figura a seguir e explique porque cada regra foi aplicada em cada item.

Regra 1: Seletor de Elemento

li { color: red; }

🔹 Peso: Baixo (seletor genérico de elemento).

✔ Aplica-se a todos os itens da lista.

Exemplo: Todos os itens da lista ficam vermelhos, a menos que outra regra com mais peso os sobrescreva.

Regra 2: Seletor de Classe

.churras { color: green; }

🔹 Peso: Médio (classe tem mais prioridade que elemento).

✔ Afeta todos os itens da lista que possuem a classe churras.

Exemplo: Os itens "Carne" e "Linguiça" deveriam ficar verdes, mas outra regra pode interferir.

Regra 3: Outra Classe com Prioridade

.fruta { color: gray; }

🔹 Peso: Médio (mesma prioridade da Regra 2).

✔ Afeta todos os itens que possuem a classe fruta.

Exemplo: "Melão", "Pera", "Uva" e "Maçã" ficam cinza.

Regra 4: Combinador Adjancente + Classe

li + li.churras { color: black; }

🔹 Peso: Alto (elemento + classe + combinador).

✔ Aplica-se apenas aos itens .churras que vêm logo após outro li.

Exemplo: "Carne" e "Linguiça" ficariam pretos, pois são precedidos por outro li.

Regra 5: Atributo Específico

.churras[draggable="true"] { color: magenta; }

🔹 Peso: Alto (classe + seletor de atributo).

✔ Afeta elementos da classe churras que possuem o atributo draggable="true".

Exemplo: Apenas "Pão de alho" fica magenta, pois é o único que tem draggable="true".

Regra 6: IDs Específicos

#lip1, #lip2, #lip3 { color: blue; }

🔹 Peso: Muito Alto (IDs têm prioridade sobre classes e elementos).

✔ Aplica-se aos itens com IDs lip1, lip2 e lip3.

Exemplo: "Pera" e "Uva" ficarão azuis, independentemente de outras regras aplicadas.

Regra 7: ID + Classe (Máxima Especificidade)

#lip3.fruta { color: orange; }

🔹 Peso: Extremamente Alto (ID + Classe combinados).

✔ Aplica-se apenas ao item que possui ID lip3 e classe fruta.

Exemplo: "Uva" ficará laranja, pois essa regra tem mais peso do que a Regra 6.

Regra 8: Elemento + Classe + ID (Inválido)

ol > li.churras#lip5 { color: purple; }

🔹 Peso: Máximo, mas sem efeito.

❌ Essa regra não afeta nenhum elemento, pois não existe um item com o ID lip5.

Exemplo: Nenhuma alteração visual ocorre.
