Explicação Detalhada das 8 Regras CSS

Regra 1: li { color: red; }

Peso: Baixo (seletor de elemento).

Aplica em: Todos os <li>.

Exemplo: Item 1 ("Abacaxi") fica vermelho (regra base).

Regra 2: .churras { color: green; }

Peso: Médio (classe).

Aplica em: Itens 6, 7 e 8 (classe "churras").

Exemplo: Itens 6 e 7 seriam verdes, mas a regra 4 prevalece.

Regra 3: .fruta { color: gray }

Peso: Médio (classe).

Aplica em: Itens 2, 3, 4 e 5 (classe "fruta").

Exemplo: Item 2 ("Melão") fica cinza.

Regra 4: li + li.churras { color: black; }

Peso: Alto (elemento + classe + combinador).

Aplica em: Itens 6 e 7 (li.churras após outro li).

Exemplo: Item 6 ("Carne") fica preto.

Regra 5: .churras[draggable="true"] { color: magenta; }

Peso: Alto (classe + atributo).

Aplica em: Item 8 (tem draggable="true").

Exemplo: Item 8 ("Pão de alho") fica magenta.

Regra 6: #lip1, #lip2, #lip3 { color: blue }

Peso: Altíssimo (ID).

Aplica em: Itens 3 e 4 (IDs #lip2 e #lip3).

Exemplo: Item 3 ("Pera") fica azul.

Regra 7: #lip3.fruta { color: orange }

Peso: Máximo (ID + classe).

Aplica em: Item 4 (ID #lip3 + classe "fruta").

Exemplo: Item 4 ("Uva") fica laranja.

Regra 8: ol > li.churras#lip5 { color: purple }

Peso: Máximo, mas não se aplica.

Motivo: Não existe elemento com id="lip5" no HTML.

Exemplo: Nenhum item é afetado.
