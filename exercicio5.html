Explicação do Exercício A

  • • Este layout vertical utiliza as seguintes propriedades CSS:
  • • display: flex no container principal para criar um layout flexível
  • • flex-direction: column para empilhar os elementos verticalmente
  • • height: 25vh em cada caixa para dividir igualmente a altura da tela
  • • box-sizing: border-box para cálculo preciso das dimensões
  • • Centralização do texto usando align-items e justify-content
  • • As cores são definidas individualmente usando nth-child() para manter a fidelidade com o exercício original
  • • A altura total do container é garantida pelo min-height: 100vh na seção principal.

Explicação do Exercício B

  • • O layout horizontal é construído com:
  • • display: flex no container para alinhamento horizontal
  • • width: 25% em cada caixa para divisão igual da largura
  • • height: 225px fixo mantendo a proporção original
  • • line-height: 225px para centralização vertical do texto
  • • Gerenciamento de espaçamento com box-sizing
  • • A combinação de flex: 1 e width: 25% garante que as caixas mantenham proporções idênticas mesmo em diferentes resoluções de tela
  • • O container principal tem altura fixa para manter a fidelidade com o exercício proposto.

Explicação do Exercício C

  • • Este exercício emprega posicionamento avançado:
  • • position: absolute para elementos nos cantos
  • • position: fixed na caixa central para posicionamento relativo à viewport
  • • transform: translate() para centralização precisa
  • • Gerenciamento de sobreposição com z-index
  • • Coordenadas top/left/right/bottom para alinhamento
  • • A caixa roxa usa position: fixed com transformação de tradução para centralização perfeita, mantendo-se visível durante o scroll
  • • As demais caixas usam posicionamento absoluto relativo ao container pai com position: relative.
